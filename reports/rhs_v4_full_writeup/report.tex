\section{Overview}
This document summarizes the topic-discovery and measurement pipeline we applied to zoning-related newspaper text. We updated the prompt used to label discovered text clusters with mechanism-specific, human-readable descriptions, and then assembled a city-by-month panel of right-hand-side (RHS) measures from the labeled text.
\begin{itemize}
\item \textbf{Topic discovery:} cluster frequency over time and UMAP scatter plots.
\item \textbf{City-by-time measures:} dashboards, event-candidate plots, and regime timelines (Sample A).
\end{itemize}
\section{Pipeline overview (high level)}
\begin{itemize}
\item Each newspaper issue is split into chunks (\~3,000 characters).
\item Chunks are embedded (OpenAI text embeddings) and clustered (UMAP $\rightarrow$ HDBSCAN) within three corpora: full ordinances, substantial amendments, targeted amendments.
\item Each discovered cluster is labeled by a language model using a prompt designed to produce mechanism-specific, human-readable labels.
\item Cluster labels are also mapped into a structured \emph{mechanics schema}: action tags (e.g., rezoning vs text amendment), dimensions (e.g., setbacks, parking), instruments, decision bodies.
\item We aggregate mechanics tags into a city-by-month panel and construct multiple RHS summaries: level (step-function) and change (jump intensity, regime switches).
\end{itemize}

\section{City-by-month RHS measures (what we built)}
We construct a canonical city-by-month panel (Sample A shown below) with multiple interpretable RHS views:
\begin{itemize}
\item \textbf{Mechanics shares (level):} city-month shares of action tags and dimension tags, aggregated from labeled clusters.
\item \textbf{Step-function state:} forward-filled between observed months (motivated by policy being piecewise-constant between updates).
\item \textbf{Semantic change intensity:} embedding-based jump measure that spikes when the text content shifts sharply.
\item \textbf{Mechanics change intensity:} PCA-based jump on the mechanics-tag vectors (spikes when the mechanism mix shifts).
\item \textbf{Policy regimes:} discrete types (KMeans on PCs) that produce regime-switch event candidates.
\end{itemize}
